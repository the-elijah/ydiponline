# ydiponline

A responsive website built with HTML5, CSS3, and JavaScript featuring Parallax scrolling and smooth navigation.

## Features
- Pure HTML/CSS/JS (no heavy frameworks)
- Parallax scrolling effects
- Smooth global scrolling
- Responsive layout with mobile navigation
- Integration with Picsum Photos for dynamic placeholders

## Development
This project is a static site. You can serve it using any static file server (e.g., `python -m http.server 8080`).
